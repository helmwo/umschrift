Kodierung von Umschrift in Unicode

Die Grundidee ist die Nutzung der mathematischen Symbol-Zeichen in Unicode.[1] So wird
    ^ra-ms-sw-HqA-^iwnw ^wsr-^mAa.t-^ra-mr.j-^imn
zu
    𝑅ꜥ-𝑚𝑠-𝑠𝑤-ℎ̣𝑘̣ꜣ-𝐼͗𝑤𝑛𝑤 𝑊𝑠𝑟-𝑀ꜣꜥ.𝑡-𝑅ꜥ-𝑚𝑟.𝑗-𝐼͗𝑚𝑛
Die Schrägstellung der Zeichen wird dabei allein durch Unicode erreicht. Der Vorteil so formatierter Dokumente ist, dass sie besser nach Umschrift durchsucht werden können.
ASCII[2] Unicode

    [1] In Unicode® 9.0.0 sind zur Zeit nur 4 Zeichen spezifisch für Umschrift definiert.
    [2] Die Umschrift-Kodierung in ASCII folgt dem MdC, das durch Programme wie Glyph und WinGlyph bekannt ist. Als Erweiterung werden Großbuchstaben durch ein vorgestelltes "^"-Zeichen markiert.
